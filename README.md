# WallAligner

  G-Earth extension for Habbo that lets you align and arrange wall items.

  ## Features

  - Visual room canvas
  - Drag and move wall items
  - Text writer with custom fonts
  - SVG/path data loader
  - Free draw mode with shapes
  - Auto adjust height
  - Place on top layer

  ## Usage

  1. Start G-Earth and connect to Habbo
  2. Run WallAligner
  3. Enter a room
  4. Select wall items and drag to move, or use the tools to create patterns
